## 🛠️ Nasıl Güncellenir?
1. Dosyaları yerel bilgisayarınıza kopyalayın:
   ```bash
   git clone https://github.com/<kullanıcı-adınız>/<repo-adı>.git
   ```
2. Değişiklikleri yapın ve geri yükleyin:
   ```bash
   git add .
   git commit -m "Değişiklik mesajı"
   git push origin main
   ```
